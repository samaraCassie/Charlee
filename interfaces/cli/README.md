# Charlee CLI

Interface de linha de comando para o Charlee (V1).

## Instalação

```bash
cd interfaces/cli
pip install -r requirements.txt
```

## Uso

```bash
python main.py
```

## Comandos (Planejado)

- `charlee inbox` - Inbox rápido
- `charlee add "tarefa"` - Adicionar tarefa
- `charlee list` - Listar tarefas
- `charlee status` - Status dos Big Rocks
- `charlee chat` - Conversar com Charlee
